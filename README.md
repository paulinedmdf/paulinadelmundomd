# paulinadelmundomd

Personal portfolio website for **Paulina Del Mundo Del Fierro, MD, MPH** — physician and clinical data scientist.

## Structure

- `index.html` — single-page site (hero, about, capabilities, experience timeline, education, publications, leadership, contact)
- `styles.css` — visual design (serif/sans typography, teal + warm paper palette, responsive timeline + cards)
- `script.js` — footer year + mobile nav toggle

## Running locally

Open `index.html` directly in a browser, or serve the folder with any static server, e.g.:

```
python -m http.server 8000
```

Then visit http://localhost:8000.

## Deploying

The site is fully static — deploy the folder as-is to GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
